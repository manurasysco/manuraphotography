# MANURA Photography — Version 2

## Cloudflare Pages settings
- Framework preset: None
- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: leave empty

## Add your photos
Upload JPG or WebP files into:
- `public/photos/wildlife/`
- `public/photos/nature/`
- `public/photos/landscapes/`
- `public/photos/travel/`
- `public/photos/portraits/`

The starter expects:
- `wildlife/wildlife-1.jpg`
- `wildlife/wildlife-2.jpg`
- `nature/nature-1.jpg`
- `nature/nature-2.jpg`
- `landscapes/landscape-1.jpg`
- `landscapes/landscape-2.jpg`
- `travel/travel-1.jpg`
- `portraits/portrait-1.jpg`

Recommended exports: JPG/WebP, 2200–2800 px long edge, quality 75–85%.
